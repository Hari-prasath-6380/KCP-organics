💡 Quick Start

First Time Setup:

1. If you see ❌ "Client not ready", restart the server

2. Watch for a QR code in the server terminal (appears in first 30 seconds)

3. Open WhatsApp on your phone → Settings → Linked Devices → Link a Device

4. Scan the QR code

5. You should see ✅ "CLIENT READY" in the server logs

6. Try this status checker again
